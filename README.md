## Getting Started

1. Review Azure DevOps and sign up for an account in the following link:  
    [https://azure.microsoft.com/en-us/pricing/details/devops/azure-devops-services/](https://azure.microsoft.com/en-us/pricing/details/devops/azure-devops-services/)

2. The free tier will be more than enough in this course, so click on any of the ‘start free’ buttons  
   ![image](https://github.com/deuspaul/AgenticAiForDevSecOps/images/gs-2)

3. Next, sign up for a GitHub account (in case you don’t already have one) at:  
   [https://github.com/signup](https://github.com/signup)  
   ![image](https://github.com/deuspaul/AgenticAiForDevSecOps/images/gs-3)

4. Next, we will enable GitHub Copilot.  
   Click on your profile at the top right corner and next click on ‘Copilot Settings’  
   ![image](https://github.com/deuspaul/AgenticAiForDevSecOps/images/gs-4)

5. Make sure ‘Show Copilot’ appears as enabled:  
   ![image](https://github.com/deuspaul/AgenticAiForDevSecOps/images/gs-5)
